# Odin Restaurant
A pure javascript restaurant website

# How to run in
1. Clone this repo
2. Run `npm install`
3. Run `npm build`
4. Double-click on `./dist/index.html`

**alternatively**

Click [here](https://mjsilva.github.io/odin-javascript-library/)
